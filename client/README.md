# Dependenices

This program uses MUI icons, Axios. To install: 

### Install Dependencies
Install Node Package Manager (npm), then run

`npm install`

Which should install all dependenices required. If `npm run dev` still does not work, try

`npm install @mui/material @emotion/react @emotion/styled @mui/icons-material axios vite` 

### Run Development Server
`npm run dev`
